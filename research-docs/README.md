# Independent Research — Track Summaries

**Project:** LMU MSBA × Cedars-Sinai AI Patient Billing Agent  
**Sprint 1 deliverable:** Four independent research tracks covering the domain
knowledge required to build and evaluate the agent.

---

| File | Track | Author |
|---|---|---|
| [billing-fundamentals.pdf](billing-fundamentals.pdf) | Healthcare billing fundamentals — revenue cycle, billing codes, billing documents, denial codes, and common patient confusion points | Sadaf Sarbazi |
| [financial-assistance.pdf](financial-assistance.pdf) | Financial assistance & FPL policy — Cedars-Sinai Charity Care, Discount Payment Plan, eligibility tiers, payment plans, California debt protections | Aziza Jamjoom |
| [ai-agent-architecture.pdf](ai-agent-architecture.pdf) | AI agent architecture — tool calling, safety hooks, RAG, prompt engineering, sub-agents, hallucination prevention | Faisal Alessa |
| [evaluation-methods.pdf](evaluation-methods.pdf) | Evaluation methods — what the system must return, labeled dataset schema design, 3-layer evaluation strategy (automated, LLM-as-Judge, human review) | Mary Tekele |

---

This research informed the synthetic validation dataset (`synthetic-data/`)
and the agent implementation roadmap (`agent-harness/`).
